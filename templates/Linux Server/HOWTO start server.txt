1. Follow "HOWTO Build Linux.txt" to build the linux server.
2. That should create a wolfded.x86 and a qagame.mp.i386.so that will both automatically copy into "templates\Linux Server" and "templates\Linux Server\main" respectively.
3. Copy mp_pak0.pk3, mp_pak1.pk3, and mp_pak2.pk3 from your RTCW install into "templates\Linux Server\main".
4. From command line, navigate to "templates\Linux Server"
5. Run the command: chmod 777 start.sh
5. Run the command: ./start.sh

You can now copy the "templates\Linux Server" folder wherever you desire to run your server.
