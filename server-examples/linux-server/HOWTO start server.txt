1. Follow "HOWTO Build Linux.txt" to build the linux server.
2. That should create a wolfded.x86 and a qagame.mp.i386.so.
3. Copy wolfded.x86 to "server-examples\linux-server"
4. Copy qagame.mp.i386.so to "server-examples\linux-server\main"
3. Copy mp_pak0.pk3, mp_pak1.pk3, and mp_pak2.pk3 from your RTCW install into "server-examples\linux-server\main".
4. From command line, navigate to "server-examples\linux-server"
5. Run the command: chmod 777 start.sh
5. Run the command: ./start.sh

You can now copy the "server-examples\linux-server" folder wherever you desire to run your server.
